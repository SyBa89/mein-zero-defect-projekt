## 📖 Über dieses Projekt

Dies ist die **Zero-Defect-Webseite** für den _Kiosk Lollipop_ in Erftstadt-Liblar.  
Es ist nicht nur eine Webseite – es ist ein **vollständiges digitales Ökosystem**, das dem Kiosk-Inhaber die volle Kontrolle über seine Online-Präsenz gibt, ohne dass er Programmierer werden muss.

---

## 🚀 Technologie-Stack

| Technologie                 | Zweck                                                            |
| --------------------------- | ---------------------------------------------------------------- |
| **Next.js 16 (App Router)** | Modernes React-Framework mit SSR und optimierten Core Web Vitals |
| **TypeScript 5.7**          | Strikte Typsicherheit für fehlerfreien Code                      |
| **Tailwind CSS 3.4**        | Utility-First CSS für konsistentes, responsives Design           |
| **Vercel**                  | Serverless-Hosting mit automatischen Deployments                 |
| **Husky + lint-staged**     | Automatische Code-Qualitätsprüfungen bei jedem Commit            |
| **ESLint + Prettier**       | Einheitlicher Code-Stil und Fehlerprävention                     |

---

## ✨ Features (Die "Krone")

- 🏠 **Modulare Startseite** – Jede Sektion ist eine eigene Komponente (Hero, Features, Produkte, etc.).
- 📦 **Hermes Paketshop-Integration** – Inklusive Paketgrößen-Rechner.
- 🛒 **Dynamisches Admin-Cockpit** _(optional)_ – Inhaber kann Notfall-Banner und Aktions-Texte live schalten.
- ♿ **Barrierefrei (WCAG AAA)** – Volle Screenreader-Unterstützung und Tastatur-Navigation.
- 📱 **Mobile-First** – Optimiert für Daumen-Zone und Touch-Interaktion.
- 🔒 **Sicherer Mitarbeiter-Bereich** – Passwortgeschützter Bereich mit Checkliste und Notizen.
- ⭐ **Google Bewertungs-Integration** – Echter Social-Proof mit 5,0-Sterne-Badge.
